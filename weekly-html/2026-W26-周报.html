二、教辅行业动态（含与各地教育部门深度合作）

  • 事件：浙江、山东 等多省教育部门与本地教辅出版社推进 “中考命题辅导” 合作项目——浙江教育出版社 与 浙江省教育厅教研室 联合开发 2026版《中考复习指南》 系列；山东教育出版社 承接 济南市初中毕业年级学业水平考试模拟题库建设。
    • 来源平台：浙江出版联合集团官网（2026-06-16）· 原文；山东教育新闻网（2026-06-17）· 原文
    • 要点：
      • 浙江：《中考复习指南》 由省教研室统稿，覆盖语数英科社五科，配套 线上刷题与错题分析系统，首批 6月20日 出货至 杭州、宁波 试点校。
      • 山东：济南市中考模拟题库 基于本地 2026年中考方案（总分调整至 760分，体育 60分），由山东教育出版社提供 数字化题库管理与成绩分析 服务。
    • 影响判断：
      • B端：区域化、定制化 中考教辅+数字化服务 模式加速，或成为省级出版集团 政府采购 新增长点。
      • C端：试点校学生可提前获得 精准模拟题，非试点地区家长 跟风购买 同类产品意愿增强，需防 版本错配。
  • 事件：河南省教育厅 与 大象出版社 签订 “基础教育数字资源共建” 战略协议，覆盖 语文、数学、英语 三科 7-9年级 数字教材与同步练习。
    • 来源平台：河南省教育厅官网（2026-06-14）· 原文
    • 要点：
      • 协议期 3年，大象出版社负责内容编制与技术对接，省厅提供 教学需求调研 与 试点学校（50所）。
      • 数字资源将通过 河南省基础教育资源公共服务平台 免费向试点校师生开放，2026年秋季学期 起试用。
    • 影响判断：
      • 大象出版社 省内渠道壁垒 得以强化，其他民营教辅公司进入河南校园的 数字端口收窄。
      • 试点校反馈将影响后续全省推广，民营教辅企业可关注 内容授权 或 技术赋能 合作机会。

三、K12教育信息与K12教育政策全国盘点

  • 事件：2026年中考 进入 6月下旬 集中考试期，多省发布 中考考务工作通知 与 命题导向说明。
    • 来源平台：中国教育报（2026-06-15）· 原文
    • 要点：
      • 山西、安徽：6月20-22日 举行，物理、化学实验操作 纳入总分（山西 15分，安徽 15分），信息技术 考查以 上机操作 方式进行。
      • 湖南：6月18-20日 考试，新增 生命安全教育 笔试（20分）纳入道德与法治科目。
      • 教育部基础教育司 强调：命题严格依标，减少死记硬背和 “机械刷题” 得分现象，增加 情境化、探究性 试题比例。
    • 影响判断：
      • 教辅企业需 紧急调整 考前冲刺卷的题型结构，增加 实验操作步骤图析 与 安全教育情境阅读。
      • 考后 真题解析 与 估分卷 需在 6月22日 后集中发布，准备时间窗口极短。
  • K12教育政策全国盘点（本周口径）：
    • 义务教育：广东、湖南、河南 多地发布 2026年中小学招生工作通知，重申 免试就近入学，严禁 “密考”“拔尖选生”，深圳龙华 报名于 6月17日 截止，后续为 学校初审 与 录取 阶段。 · 原文
    • 普通高中：江西、湖北 等地陆续公布 2026年普通高中招生计划，省级示范高中 指标到校比例不低于 70%。
    • 特殊教育：教育部 联合 中国残联 发布 《关于加强普通学校随班就读工作的指导意见》，明确 5人以上 残疾学生的学校须配备 资源教室。
    • 对我方映射：中考 相关教辅进入 最后冲刺出货期，客服须掌握各省考试时间与科目变化。

四、服务K12的平台动态

  • 事件：猿辅导、作业帮 暑期新品上市，主打 AI学习机 与 精准学 功能。
    • 来源平台：微信公众号「教育科技观察」（2026-06-16）· 原文；B站教育UP主「数码测评老王」（2026-06-17）· 原文
    • 要点：
      • 猿辅导AI学习机2.0：首发 “AI精准学Pro”，支持 多模态错题识别（题目、图形、手写公式），并自动生成 同知识点3道变式题；6月18日 开启预售，定价 4299元。
      • 作业帮智能练习灯T30：内置 “AI朗读助手” 与 “中英文作文智能批改” 功能，6月20日 起在 京东、天猫 首发。
    • 影响判断：
      • 暑期 智能硬件 市场争夺战 提前打响，与 传统教辅 形成 “线上+线下” 分流。
      • 内容侧：出版社可探索 AI辅导硬件的内置内容授权（如标准朗读音频、分级作文库），拓展数字营收渠道。
  • 事件：百度文库 升级 “初高中AI学习助手”，接入 大模型 提供 试题解析与知识点讲解。
    • 来源平台：36氪（2026-06-15）· 原文
    • 要点：
      • 基于 文心大模型4.5，支持 一拍即讲（题目+解题步骤+知识点关联），覆盖 初中语数英、物化生 主流科目。
      • 对 百度文库VIP用户 免费开放，普通用户每日可体验 5次。
    • 影响判断：
      • 以 免费+轻度付费 模式冲击 传统拍照搜题App 市场（如小猿搜题），后者或被迫 压缩会员价格 或在 内容深度 上差异化。
      • 教辅企业 内容版权授权 需要关注：避免被 未经授权的AI解析 所使用。

五、政府政策与通知（含K12教辅政策全国盘点）

  • 政策：广东省中小学教辅材料评议实施细则（2026年修订）
    • 发布单位：广东省教育厅
    • 发布时间：2026-06-16
    • 原文链接：广东省教育厅官网
    • 核心条款：
      • 教辅实行 “一科一辅”，学校不得统一代购目录外教辅。
      • 新增 数字教辅（App、在线题库等）需经省 教辅评议委员会 审查，审查纳入年度评议周期。
      • 教辅定价实行 政府指导价，纸质教辅 利润率不得超过5%，数字教辅 按学期定价。
    • 执行影响：
      • 数字教辅纳入监管，之前 游离于评议外的AI题库 需进入 合规通道。
      • 利润率5% 的限制，将倒逼出版社 降低成本 或 向规模化/衍生服务 转型。
      • 广东作为 教辅消费大省，此举可能被 浙江、江苏 等省份借鉴。
  • 政策：2026年度全国教育科学规划项目申报
    • 发布单位：教育部办公厅
    • 申报截止：2026年6月25日
    • 原文链接：全国教育科学规划领导小组办公室
    • 核心条款：课题方向设置 “人工智能教育出版标准”、“数字教材的课堂应用成效” 等教育出版相关选题。
    • 执行影响：教辅出版社 研究部门 可联合高校申报，争取 产业政策话语权。
  • K12教辅政策全国盘点（本周口径）：
    • 教材教辅监管：广东 新规为本周重点；江苏、山东 等待各地贯彻 2026年秋季学期教辅目录 的调整动态。
    • 校外培训监管：教育部校外教育培训监管司 发布 《关于做好2026年暑期校外培训治理工作的通知》，严查 学科类隐形变异 培训；考后提分班、衔接班 为重点打击对象，教辅推广避免 “课后/周末提优” 话术。 · 原文

七、出版社/教辅公司：出版及数智化方向最新动态

  • 公司/机构：人民教育出版社——人教智慧教育平台 上线新板块
    • 本周动态：6月16日，人教社在 “人教智慧教育”平台 上线 “暑期托管·素养拓展” 专题资源，包含 科学实验、艺术鉴赏、编程入门 等非学科类内容，面向 1-6年级 学生免费开放。
    • 来源平台：人民教育出版社官网（2026-06-16）· 原文
    • 影响判断：顺应 暑期托管需求，打造 官方非学科内容的轻量级入口；对民办教辅机构 同类素养课 形成 “免费替代” 压力。
    • 可跟进点：是否在 2026秋季学期 开放 学科类补充资源 收费模块。
  • 公司/机构：世纪天鸿 发布 2026年半年度数字教辅运营数据
    • 本周动态：6月15日，世纪天鸿（300654）在深交所互动易平台透露：其 “智慧教辅” 系列产品（含 数字化自主练习、智慧课堂）已覆盖 全国300余所 学校，2026年Q2 新增 40余所 学校。
    • 来源平台：深交所互动易（2026-06-15）· 原文
    • 影响判断：B端进校模式 是传统教辅公司数字化转型的 主要路径，但 C端用户付费转化 仍是挑战。
    • 可跟进点：关注其 2026年中报 中数字业务营收占比。

十、风险与机会清单

  • 风险：
    • 中考估分 相关产品若出现 严重偏差，可能引发 退单与投诉，需附 免责声明 并引导至官方查分系统。
    • 广东教辅新规 落地若被误读，出版社自行 扩大合规范围 可能导致 审批滞后与渠道断货；法务解读 与 省厅咨询热线 需及时对齐。
    • 暑期AI学习机 广告 夸大“提分”效果，可能被 市场监管部门 点名处罚，教辅企业 借势营销 时应保持克制。
    • 志愿填报“保录”“大数据预测” 类商业网站/公众号被 集中查处，教辅类 志愿指南 推广须严守 “工具书，不提供预测” 底线。
  • 机会：
    • 中考实验操作 权重提升：可 快速开发《物理/化学实验操作步骤卡》 口袋书 或 微课视频，在 初二/初三家长社群 中推广。
    • 书博会 馆配客户 集采需求：7月10日前 完成 重点书目 的 馆配函 与 折扣方案 寄送；关注 图书馆馆长、政教主任 等关键人现场到访时间。
    • 智慧阅读教室 解决方案：出版社可联同 本地教育装备代理商，向 区县教育局 推 “阅读提升工程”，将 纸质书销售 升级为 “图书+平台+培训” 打包服务。
    • 广东新政 下，率先 通过 数字教辅评议 的出版社，在 广东市场 将获得 准入门槛带来的先发优势；建议 Q3 优先提交报审。
