# 教辅行业与K12动态周报中心

> 用途：累计沉淀每周动态，固定在同一页面查看。  
> 更新频率：北京时间每周一 10:05 起由 GitHub Actions 自动生成、校验、构建并推送部署。

## 使用说明

1. 每周自动创建 `weekly/YYYY-WW-周报.md`，并把最新周报链接插入本页顶部索引。
2. 周报正文只保留有可靠来源的板块；没有信息的板块直接省略。
3. 每条信息固定为：标题 + 简短介绍 + 原文链接。
4. 历史周报长期保留，不覆盖、不删除。

## 自动化（GitHub → Vercel）

- 工作流：`.github/workflows/weekly-autopublish.yml`（北京时间周一 10:05 / 10:20 / 10:35 运行，12:05 / 14:05 备用）。
- 也可在 GitHub Actions 中手动点 **Weekly skeleton and publish → Run workflow**。
- 强制重写当周正文：Actions → **Force AI refill (当前周报)** → **Run workflow**。
- 仓库 **Settings → Actions → General → Workflow permissions** 须为 **Read and write permissions**，否则无法自动推送。
- 正文优先使用 AI 填写；没有 `ANTHROPIC_API_KEY` / `OPENAI_API_KEY` 或 AI 调用失败时，会用官网摘录与官方入口生成兜底正文，避免整周缺页。

## 本机命令

- 完整生成当周周报并构建：`npm run weekly:create`
- 仅生成骨架：`npm run weekly:skeleton`
- 仅抓取摘录：`npm run weekly:digest`
- 仅填正文：`npm run weekly:fill`
- 校验正文格式：`npm run weekly:validate`
- 仅生成 HTML：`npm run build`

## 正文板块（固定）

- 一、K12教育政策
- 二、K12教辅政策
- 三、出版数智化
- 四、局社合作
- 五、科技合作
- 六、评教辅行业
- 七、政策解读

## 采编口径

- 正文只写本周采编窗口内发布/更新的新闻，或持续有效的官方入口。
- 旧官宣、旧预备会、旧活动报道不得作为本周新闻写入。
- “评教辅行业”只收新闻评论、行业媒体评论、专家评论、专栏文章或署名评论文章。
- “政策解读”只收政策解读、问答、图解、一图读懂、专家解读等解读类文章。
- 没有可靠来源时直接省略该条或该板块，不写占位说明。
- 正文不得出现“来源平台、来源、要点、影响判断、建议动作、风险、机会”等字段。

## 信息来源

- 官方网站：教育部、各省市教育厅/教育局、国家新闻出版署、中国政府网等。
- 行业媒体：人民日报、新华社教育频道、中国教育报、中国出版传媒商报、教育装备网等。
- 出版机构：出版集团、出版社、教辅公司官网及公开公众号/服务号。
- 社媒与自媒体：微信公众号/服务号、B站、小红书、微博等公开可核验来源。
- 公众号/服务号如需自动采集，可在 `config/weekly-rss.json` 配置可访问 RSS 地址；正文仍只输出标题、简短介绍和原文链接。
