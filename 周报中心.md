# 教辅行业与K12动态周报中心

> 用途：累计沉淀每周动态，固定在同一页面查看。  
> 更新频率：每周一 10:00（GitHub Actions 云端自动推送，或本机计划任务 / 手动脚本）。

## 使用说明

1. 每周复制 `templates/周报模板.md`，命名为：`weekly/YYYY-WW-周报.md`。  
2. 在本页顶部的“周报索引”新增一行链接（最新放最上面）。  
3. 保留历史周报，不覆盖旧内容，实现长期留存。  

### 自动化（GitHub → Vercel，推荐）

- 工作流：`.github/workflows/weekly-autopublish.yml`（北京时间每周一 10:00 运行；也可在 Actions 里 **Run workflow** 手动测）
- 仓库 **Settings → Actions → General → Workflow permissions** 须勾选 **Read and write permissions**，否则无法 `git push` 触发线上更新。
- 等价本机命令（需 Node.js）：`npm run weekly:create`（生成当周骨架、更新本页索引、`npm run build`）

### 自动化（Windows 本机，可选）

- 首次注册计划任务（每周一 10:00 自动执行，逻辑同 `npm run weekly:create`）：
  - `powershell -ExecutionPolicy Bypass -File .\scripts\register_weekly_task.ps1`
- 手动立即执行一次：
  - `powershell -ExecutionPolicy Bypass -File .\scripts\create_weekly_report.ps1`
- 仅生成 HTML（与 Vercel 部署一致）：
  - `npm run build`
- HTML查看入口（自动生成）：
  - `周报中心.html`
- 线上部署（GitHub + Vercel）说明：
  - 见 `docs/部署到GitHub与Vercel.md`

## 信息范围（固定口径）

### 1) 教辅行业动态（含与各地教育部门深度合作）
- 教辅出版公司合作、渠道动作、重点产品
- **与各地教育部门（省/市/县）及事业单位的战略合作、资源共建、课题/试点落地**
- 行业展会、论坛、订货会、区域活动
- 招投标、战略签约、并购与资本动向
- 会务计划（近期时间、地点、主办方、参会建议）
- **出版社/教辅公司出版与数智化方向的最新动态**：线索可从行业与公司披露中采集，**周报应答落在「九」**（与本条行业动态互证）
- **出版社/教辅公司与科技公司的深度合作**：**周报应答落在「十」**（与文旅、硬件等跨行合作合并归类）

### 2) K12教育整体信息与K12教育政策全国盘点
- 学段需求变化（小学/初中/高中）
- **K12教育政策全国盘点**：国家新规与部委口径、省级重点变化、执行窗口与舆情敏感点
- 重点省市政策执行差异与市场影响
- 学校端教研、考试、数字化建设趋势

### 3) 服务K12的平台动态
- 教育信息化平台、资源平台、测评与题库平台
- AI+教育产品更新、接入与商业化动作
- 典型平台合作案例与可复用打法

### 4) 政府政策与通知（含K12教辅政策全国盘点）
- 教育部及各省市教育厅/教委新规
- “双减”、课后服务、考试招生相关政策
- 数字教育、教材教辅监管、数据合规要求
- **K12教辅政策全国盘点**：教辅评议与选用、进校园管理、线上线下销售合规、地方细则与典型案例

## 每周更新流程（建议）

- 周一 10:00 收集信息源（政策官网、行业媒体、平台公告、展会官号）
- 10:00-10:20 去重与分级（高/中/低优先级）
- 10:20-10:40 输出“本周要点 + 影响解读 + 行动建议”
- 10:40-10:50 写入当周周报并更新本页索引
